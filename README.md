## What it does

Fortnite Fight the BRAINROT Power Calculator is a single-page HTML tool for estimating combat power in Fortnite's Fight the BRAINROT mode. It is designed for fast entry, immediate output, and browser-only usage, so you can check results without setting up a server or installing extra software.

It also includes reverse calculation, which is useful when you already know the power target and want to infer possible input values. Mutation handling is part of the calculation path as well, making the page practical for tuning builds and seeing how different inputs shift the final number.

## Script Features

- Includes mutation parameter support
